## About this model

This is a **low-poly 3D model** of my experimental rocket.  
The real version is under construction.

**Propellant:** Potassium nitrate + sugar (R-Candy / rocket candy)  
**Nozzle:** Movable (for thrust vectoring experiments)  
**Body:** Reusable tube – lightweight, reinforced  
**Status:** Early prototype – breaking sound barriers step by step

The 3D model was created in Blender.  
It's a work in progress – just like the real rocket.

---

## Tech

- Modeled in **Blender 4.0**  
- Exported as **GLTF / GLB**  
- Ready for web, AR, or 3D printing
